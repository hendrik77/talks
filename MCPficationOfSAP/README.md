# MCP'fication of SAP

Presentation about how the Model Context Protocol (MCP) is transforming SAP development. Covers MCP basics, the SAP TechEd 2025 announcements, official SAP MCP servers (CAP, Fiori, UI5, MDK), community-built servers (SAP Docs, ABAP ADT, OData Bridge), real-world use cases, and the outlook for the agentic enterprise.

## Running the slides

Requires [Node.js](https://nodejs.org/).

```bash
npm install
npm run dev
```

Then open <http://localhost:3030>.

## Building & exporting

```bash
npm run build       # static SPA in ./dist
npm run export      # export to PDF
```
